# 启动应用

启动黑马记账APP的开发服务器，方便快速预览和调试。

## 执行步骤

1. 在项目根目录执行启动命令：
   ```
   npx vite --port 3000
   ```
2. 启动后自动在浏览器打开 **http://localhost:3000**

## 说明

- 使用 Vite 开发服务器，支持热更新（改代码自动刷新页面）
- 端口固定为 **3000**，避免与其他服务冲突
- 数据存储在浏览器 localStorage 中，启动即可使用，无需额外配置数据库
